- 转账接口：`/api/transfers` 已从占位改为真实列表、详情、创建、编辑、删除。

- 搜索筛选：前端账单搜索页已接入后端筛选能力。
- 账户管理：新增、编辑、归档、计入资产、资产/负债切换、资产趋势概览。
- 报表：分类、趋势、对比、成员统计均已接入真实接口。

- 预算：已有月总预算持久化，首页预算气泡和“更多 - 预算设置”可调整。
- 备份恢复：创建、列表、下载、从备份恢复、恢复前自动留档；前端已有入口。
- 群晖部署文件：Dockerfile、根 compose、`deploy/synology/docker-compose.yml`、备份/恢复脚本、部署文档。
